# CardCollector

**CardCollector** is a lightweight command-line application designed for trading card enthusiasts to manage their collections quickly and efficiently.

If you prefer typing over clicking, CardCollector allows you to track the cards you own, organise them with tags, search using multiple attributes, and monitor your collection history — all from your terminal.

## Key Features
* **Inventory Management**: Add, edit, and remove cards by index or name.
* **Wishlist Support**: Maintain a separate wishlist and transfer cards into your inventory when acquired.
* **Granular Tracking**: Track card details such as name, price, quantity, set, rarity, condition, language, card number, notes, and tags.
* **Advanced Search & Filtering**: Search using multiple attributes (name, metadata, notes, tags) or quickly filter by tag.
* **Tagging System**: Organise cards using lightweight tags (e.g. `deck`, `trade`, `sealed`).
* **Duplicate Detection**: Identify cards with identical variants in your collection.
* **Sorting & Reordering**: Sort and reorder your collection based on different criteria.
* **Analytics**: View insights such as total value and most valuable cards.
* **History Logs**: Track when cards were added, modified, or removed.
* **Undo Support**: Revert recent changes to prevent mistakes.
* **Import & Export**: Save and restore your collection using upload/download.
