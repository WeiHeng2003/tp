# CardCollector

CardCollector is a lightweight command-line application for trading card enthusiasts to manage their collections quickly and efficiently.

If you prefer typing over clicking, CardCollector lets you track cards, search your inventory, and review collection history from your terminal.

For the full product overview and feature list, see the documentation below.
